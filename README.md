# Project Documentation

## Overview
This project is a system that provides access control and management features for users and resources.

## Setup Instructions
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/dominikmoravek/pristupovy-system.git
   cd pristupovy-system
   ```
2. **Install dependencies:**  
   ```bash
   npm install
   ```
3. **Configure environment variables:**  
   Create a `.env` file in the root directory and add the necessary configurations:
   ```bash
   DATABASE_URL=your_database_url
   SECRET_KEY=your_secret_key
   ```
4. **Run the application:**  
   ```bash
   npm start
   ```

## Usage
After the application is running, you can access the user interface and start managing resources and permissions.
